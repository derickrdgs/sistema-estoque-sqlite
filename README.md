Sistema de Estoque com Banco de Dados (Java + SQLite)

Aplicação desenvolvida em Java com integração a banco de dados SQLite.

Funcionalidades
- Cadastro de produtos no banco
- Listagem de produtos
- Persistência de dados

Tecnologias
- Java
- SQLite
- JDBC

Objetivo
Simular um sistema com persistência de dados utilizando banco de dados.
